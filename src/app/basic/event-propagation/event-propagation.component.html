DOM is using event propagation so when event is fired on all elements where define in it path up to the root element
  So if:
    <element-1>
      <element-2>
        <element-3>
  if element-3 event is firing then its goes up to element-2 and well as element-3.  

  To stop this propagation use in the component event handler:
  
    onClick($event) 	{
      event.stopPropagation;
    }
