Use when needed to inject a contant to the template from outside. You can use property binding but in cases where it is more complex contant use ng-contant. ng-content enable the component user to customize the content of a component.

For example the template looking like:

<div class="panel panel-default">
  <div class="panel-heading">
    // identify the ng-contant select as class="heading"
    <ng-contnat select=".heading"></ng-contant>
  </div>
  <div class="panel-body">
    // identify the ng-contant select as class="body"
    <ng-contnat select=".body"></ng-contant>
  </div>
</div>
To use this component: 
<the-component>
  <div class="heading"></div>
  <div class="body">
    <h2>This is body</h2>
  </div>    
>/the-component>
