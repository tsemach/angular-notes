There are two ways of adding bootstrap support
  1. Adding into angular-cli.json
  2. Adding into style.css
Option-1: using angular-cli.json 
  # npm install --save bootstrap@3
    
  // open the file .angular-cli.json and add the boostrap css file in it
  -) under the section "styles": [ ... ]
  -) add the entry "../node_modules/bootstrap/dist/css/bootstrap.min.css"
Option-2: edit the file sytle.css
  • run: npm install bootstrap --save
  • edit style.css add the line: @import "~bootstrap/dist/css/bootstrap.css;"
