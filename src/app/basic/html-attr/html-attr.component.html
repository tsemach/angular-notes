Thml attribute are in most cases 1-to-1 mapping with DOM element BUT not in all cases.

Also the opposite, html attribute may not have DOM attribute

When using attribute binding we are binding into a DOM attribute (not the html).

So When need to targing HTML attribute need to tell Angular about it by adding 'attr.' prefix

For example: 

  <table>
    <tr>
      <td [attr.colspan]="colspan"> Table Data </td>
    <tr>
  </table>
