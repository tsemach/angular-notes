Passing variables from template to component can be done in two ways
  1. Passing by $event
  2. Passing by #something
Option-1: passing by $event 
  <input> (keyup.enter)="onKeyUp($event)"/>
  
  onKeyUp($event) 	{
    console.log($event.target.value);
  }
Option-2: passing by #something 
  <input> #email (keyup.enter)="onKeyUp(email.value)"/>
  #email: is a referance to input field.
  
  onKeyUp(email) 	{
    console.log(email);
  }
