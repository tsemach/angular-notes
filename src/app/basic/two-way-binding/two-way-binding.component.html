To enable two communication between a component and a view. 

Option-1: by inline in the event binding
<input> [value]="email" (keyup.enter)="email = $event.target.value; onKeyUp($event)"/>

onKeyUp() 	{
  console.log("email = ", email);
}
    

Option-2: using angular ngModel two way data binding
<input> [(ngModel)]="email" (keyup.enter)="onKeyUp($event)"/>

onKeyUp() 	{
  console.log("email = ", email);
}
